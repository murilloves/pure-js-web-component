# MyPersonalWebSite

## Production mode:

* [My Personal Website](https://murilloves.github.io/) - Running app

## Development mode:
### Install:
`npm install -g http-server`

### Serve (Run):
`http-server ./ -p 80`

### It should be Listening at:
* [http://127.0.0.1](http://127.0.0.1)

or

* [http://192.168.25.99](http://192.168.25.99)
